# CashMachine
Everi Digital's Cash Machine slot machine created in Java.

This personal project was created to practice my programming skills when I was first learning Java. No GUI, simply showcases the core gameplay in command prompt.

Cash Machine is one of my favorite slot machines - it has a unique respin system that automatically triggers when a losing spin has a "0" on any of the reels in play.
Additionally, an empty reel on a winning spin might also respin for a chance to get a larger prize.

To play, run CashMachine.class with Slots.class in the same directory.
